Friends and Foes¶

Default player list can be additionally splitted to friend and enemy section in order to be better using Alert system

adding friends and foes uses same commands as in other sections and remember that, instead of writing precise names you can insert any substring of it (player tag for example)

due to add, remove, clear being already explained in other sections they would be here without much details

friend add¶

.friend add "DSE)"

.friend add "DSE)", "RNS", "CNS"

friend remove¶

.friend remove "John"

friend clear¶

.friend clear

enemy add¶

.enemy add "MNS-"

.enemy add "MNS-", "John", "BGG-Silverstream"

enemy remove¶

.enemy remove "MNS-"

enemy clear¶

.enemy clear
