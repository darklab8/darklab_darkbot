Friends and Foes
====================

Default player list can be additionally splitted to friend and enemy section
in order to be better using :ref:`Alert` system

adding friends and foes uses same commands as in other sections
and remember that, instead of writing precise names
you can insert any substring of it (player tag for example)

due to **add**, **remove**, **clear** being already explained in other sections
they would be here without much details

friend add
************

.. code-block::

    .friend add "DSE)"

.. code-block::

    .friend add "DSE)", "RNS", "CNS"

friend remove
******************

.. code-block::

    .friend remove "John"

friend clear
******************

.. code-block::

    .friend clear

enemy add
******************

.. code-block::

    .enemy add "MNS-"

.. code-block::

    .enemy add "MNS-", "John", "BGG-Silverstream"

enemy remove
******************

.. code-block::

    .enemy remove "MNS-"


enemy clear
******************

.. code-block::

    .enemy clear
